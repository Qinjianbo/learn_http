- URI - Uniform Resource Identifier - 统一资源标识符
- URL - Uniform Resource Locator - 统一资源定位符
- URL包含于URI
- URI完整格式如下：

|协议名称|用户名密码|服务器地址|端口号|带层次的文件路径|查询字符串|片段标识符|
|:------:|:-------:|:---------:|:---:|:--------------:|:---------:|:--------:|
|http://|user:pass|@www.boboidea.com|:80|/user/profile.md|?userId=1&name=bobo|#introduction|
